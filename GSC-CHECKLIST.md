## 🔴 常見問題處理

### 問題 1：「網址不在 Google 索引中」
**解決方案**：
1. 確認網址正確（https://fun1399.com/...）
2. 檢查 robots.txt 未阻擋該網址
3. 在 GSC 點擊「要求建立索引」
4. 等待 1-3 天重新檢查

### 問題 2：Sitemap 顯示「失敗」
**解決方案**：
1. 確認 sitemap.xml 可以正常訪問
2. 檢查 XML 格式是否正確
3. 重新提交 Sitemap

### 問題 3：Core Web Vitals 顯示「需要改善」
**解決方案**：
1. 使用 PageSpeed Insights 檢測
2. 優化圖片大小
3. 啟用 Gzip/Brotli 壓縮
4. 使用 CDN
