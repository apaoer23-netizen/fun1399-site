**每篇文章必須包含的連結：**
- 連結到所屬 Pillar Page（如百家樂系列 → 百家樂總覽頁）
- 連結到 2-3 篇同系列其他文章

- 連結到 1 個 Money Page（鉅城優惠頁）

### 3. Schema 標記必裝清單

**每篇文章必須添加：**
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "文章標題",
  "author": {
    "@type": "Organization",
    "name": "鉅城娛樂城"
  },
  "publisher": {
    "@type": "Organization",
    "name": "鉅城娛樂城",
    "logo": {
      "@type": "ImageObject",
      "url": "網站 Logo URL"
    }
  },
  "datePublished": "2025-03-01",
  "dateModified": "2025-03-01"
}
```

**FAQ 文章添加：**
```json
{
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "問題",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "答案"
    }
  }]
}
```
