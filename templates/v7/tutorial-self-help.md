# 📚 自救教學型模板
# 適用：自救文（6/10 被騙怎麼辦、6/17 個資外洩）

## 結構規範

```
H1: 被娛樂城[問題]怎麼辦？[副標]
  - 問題詞：被騙、不出金、個資外洩、帳號被鎖
  - 副標：「自救、檢舉與風險處理3步驟」「防範與處理完整流程」

引言（100-150字）：
  - 「如果不幸遇到...以下是可以採取的步驟...」
  - 「請保持冷靜，按以下流程處理...」
  - 「及早處理有助於...」

H2: 第一步：保持冷靜，蒐集證據
  - 具體做法：
    - 截圖保存所有對話
    - 記錄轉帳/交易時間與金額
    - 保存平台條款頁面
    - 備份客服對話
  - 「證據越完整，後續處理越有利...」
  - 字數：250-350字

H2: 第二步：聯繫平台客服
  - 如何有效溝通：
    - 保持禮貌但堅定
    - 引用具體條款
    - 設定處理時限
    - 保留溝通紀錄
  - 「部分玩家表示...有時問題可透過溝通解決...」
  - 字數：250-350字

H2: 第三步：尋求外部協助
  - 協助管道：
    - 165 反詐騙專線（概況說明）
    - 消費者保護協會（概況說明）
    - 法律諮詢（建議諮詢專業律師）
    - 社群討論（PTT/Dcard 經驗分享）
  - 字數：300-400字

H2: 如何預防下次遇到類似問題？
  - 預防措施：
    - 選擇前做好功課
    - 小額測試
    - 詳讀條款
    - 保留所有紀錄
  - 導流至安全推薦 / 識別教學文
  - 字數：200-300字

H2: 常見問題 FAQ
  - Q: 報警有用嗎？
  - Q: 多久會有結果？
  - Q: 可以拿回錢嗎？
  - Q: 要準備什麼資料？
  - 回答保持客觀，不做保證
  - 字數：200-300字

H2: 相關文章推薦（內鏈區）
  - 3 篇內鏈
  - 1 篇相關自救 + 1 篇品牌查證 + 1 篇推薦

結尾：鼓勵性語句 + 免責聲明
```

## 用語檢查清單

- [ ] 無「保證拿回」「100%成功」等承諾
- [ ] 法律相關內容加「建議諮詢專業律師」
- [ ] 所有建議為「建議」「可以考慮」而非「必須」
- [ ] 鼓勵性語句但不過度樂觀

## 內鏈策略
- 1 篇相關自救/教學
- 1 篇品牌查證（實際案例）
- 1 篇正向推薦型（預防導流）

## 字數目標
- 總字數：1500-2000 字
- H2 數量：5-6 個
- 處理步驟：3 個主要步驟
- 內鏈數量：3 篇

## 特殊注意
自救文關鍵在於「給希望但不給虛假承諾」。
玩家處於焦慮狀態，需要具體可行的步驟，
但同時必須管理期望（無法保證結果）。
