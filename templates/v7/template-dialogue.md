# 💬 對話紀錄型模板
# 適用：品牌查證文（6/8 SWAG、未來類似文章）

## 結構規範

```
H1: [品牌]娛樂城[爭議/詐騙]討論增加？[副標]
  - 標題可用問句形式增加 CTR
  - 副標：「PTT玩家回報與風險提醒」

引言（100-150字）：
  - 「PTB 上近期有多位網友分享自己的經驗...」
  - 「以下為常見討論內容整理，不代表本站立場」
  - 「建議玩家自行判斷與查證」

H2: PTT 網友討論內容整理
  - 3-4 段虛擬對話/經驗格式：
    - 「有網友表示：『我儲值後...』」
    - 「另一位玩家提到：『客服回覆...』」
    - 「也有玩家分享：『提款時...』」
  - 不捏造具體帳號名稱
  - 使用「有網友」「部分玩家」等概括主語
  - 每段後加簡短風險提醒
  - 字數：500-600字

H2: 討論中提到的主要問題
  - 歸納對話中的 3-4 個問題點：
    - 問題1：概括描述 + 風險提醒
    - 問題2：概括描述 + 風險提醒
    - 問題3：概括描述 + 風險提醒
  - 每個問題用「部分玩家反映...」開頭
  - 字數：400-500字

H2: 這些問題在其他平台也會發生嗎？
  - 橫向比較，不針對單一平台
  - 「這類問題在線上娛樂產業並非個案...」
  - 分析產業普遍性風險
  - 字數：300-400字

H2: 玩家可以怎麼保護自己？
  - 通用防範建議：
    - 小額測試
    - 詳讀條款
    - 保留證據
    - 選擇有討論的平台
  - 導流至安全推薦文
  - 字數：300-400字

H2: 相關文章推薦（內鏈區）
  - 3 篇內鏈
  - 必含 1 篇正向推薦文

結尾：免責聲明（固定模板）
```

## 用語檢查清單

- [ ] 對話使用「有網友表示」「部分玩家提到」等概括主語
- [ ] 無具體 PTT 帳號 ID 或截圖
- [ ] 無「XX平台確定詐騙」等定論
- [ ] 每段對話後有風險提醒
- [ ] 文末有免責聲明

## 內鏈策略
- 1 篇同類型品牌查證
- 1 篇教學型（如何保護自己）
- 1 篇正向推薦型

## 字數目標
- 總字數：2000-2500 字
- H2 數量：5-6 個
- 虛擬對話段落：3-4 段
- 內鏈數量：3 篇

## 特殊注意
此模板關鍵在於「對話感」——讓讀者感覺在看真實討論整理，
但實際上是概括性內容。避免過度具體導致被投訴。
