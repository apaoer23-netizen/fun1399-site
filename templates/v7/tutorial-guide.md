# 📚 教學步驟型模板
# 適用：教學文（6/5 出金教學、6/24 黑網手法）

## 結構規範

```
H1: [主題]完整教學/解析：[副標]
  - 教學詞：教學、指南、攻略、解析
  - 副標：「如何確保提款順利」「識別高風險平台方法」

引言（100-150字）：
  - 「很多[新手/玩家]對...有疑問...」
  - 「本文整理完整的...教學...」
  - 「建議搭配實際操作學習」

H2: 什麼是[主題]？
  - 基礎說明
  - 定義 + 重要性
  - 字數：150-200字

H2: [主題]的標準流程/常見手法
  - 步驟型：
    - 步驟 1：[名稱] → 說明 → 圖示建議
    - 步驟 2：[名稱] → 說明 → 注意事項
    - 步驟 3：[名稱] → 說明 → 常見錯誤
    - 步驟 4：[名稱] → 說明 → 進階技巧
  - 解析型（黑網手法）：
    - 手法 1：[名稱] → 特徵 → 識別方法
    - 手法 2：[名稱] → 特徵 → 識別方法
    - 手法 3：[名稱] → 特徵 → 識別方法
  - 字數：600-800字

H2: 常見問題與解決方法
  - Q&A 格式：
    - Q: [常見問題1]？
    - A: [解答]
    - Q: [常見問題2]？
    - A: [解答]
    - Q: [常見問題3]？
    - A: [解答]
  - 字數：300-400字

H2: 進階技巧與注意事項
  - 給有經驗的玩家
  - 3-5 個進階提示
  - 字數：200-300字

H2: 相關文章推薦（內鏈區）
  - 3 篇內鏈
  - 1 篇相關教學 + 1 篇品牌查證 + 1 篇推薦

結尾：鼓勵性語句 + 更新日期標註
```

## 步驟型 vs 解析型差異

| 維度 | 步驟型（6/5 出金） | 解析型（6/24 黑網） |
|------|-------------------|-------------------|
| H2 | 「標準流程」 | 「常見手法」 |
| 結構 | 步驟 1/2/3/4 | 手法 1/2/3 |
| 語氣 | 教學引導 | 分析警示 |
| CTA | 「立即試試」 | 「查看安全推薦」 |
| 內鏈 | 教學延伸 | 品牌查證實例 |

## 用語檢查清單

- [ ] 步驟清楚，每步有具體說明
- [ ] 常見問題為真實玩家會問的
- [ ] 進階技巧確實有價值，非填充內容
- [ ] 無過度推銷特定平台

## 內鏈策略
- 1 篇相關教學（深入或延伸）
- 1 篇品牌查證（實際案例）
- 1 篇正向推薦型

## 字數目標
- 總字數：1500-2000 字
- H2 數量：4-5 個
- 步驟/手法：3-4 個
- 內鏈數量：3 篇
