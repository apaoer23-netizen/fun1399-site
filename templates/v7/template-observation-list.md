# 📊 名單比較型模板
# 適用：風險觀察名單（6/15 高風險觀察）

## 結構規範

```
H1: 2026年[月份]高風險娛樂城觀察：[副標]
  - 避免「黑網名單」「官方認證」等詞
  - 副標：「玩家投訴較多平台整理」「近期討論觀察」

引言（100-150字）：
  - 「我們整理近期網路討論中...」
  - 「以下為玩家投訴較多平台的觀察...」
  - 「僅供參考，不代表官方立場」
  - 「請玩家自行查證與判斷」

H2: 觀察方法說明
  - 資料來源：PTT、Dcard、社群媒體（公開討論）
  - 非官方認證、非法律判定
  - 「觀察名單僅反映網路討論熱度...」
  - 字數：150-200字

H2: 近期討論較多的平台整理
  - 表格：
    | 平台名稱 | 爭議類型 | 討論熱度 | 風險提醒 |
    |----------|---------|---------|---------|
    | XX娛樂城 | 提款問題 | 高 | 建議謹慎評估 |
    | YY娛樂城 | 客服爭議 | 中 | 注意溝通記錄 |
  - 每個平台 2-3 句客觀說明
  - 使用「部分玩家反映」「網友討論指出」
  - 5-8 個平台
  - 字數：500-700字

H2: 這些爭議的共同特徵
  - 歸納產業問題：
    - 共同問題1 + 說明
    - 共同問題2 + 說明
    - 共同問題3 + 說明
  - 不針對單一平台做結論
  - 字數：300-400字

H2: 玩家如何判斷平台風險？
  - 5 項檢查要點
  - 導流至安全推薦文 / 教學文
  - 字數：200-300字

H2: 相關文章推薦（內鏈區）
  - 3 篇內鏈

結尾：免責聲明（特別加：「本觀察名單非官方認證」）
```

## 表格差異化規則

與「推薦/排名」文差異：
- 風險觀察：無評分，只有「討論熱度」+「風險提醒」
- 推薦文：有評分、優缺點、適合誰
- 排名文：有排名、分數、詳細比較

## 用語檢查清單

- [ ] 標題/內文無「黑網名單」「官方認定」
- [ ] 表格使用「風險提醒」而非「判定結果」
- [ ] 每個平台說明以「部分玩家反映」開頭
- [ ] 文末特別聲明「非官方認證」

## 內鏈策略
- 1 篇詐騙更新/趨勢文
- 1 篇教學型（如何判斷）
- 1 篇正向推薦型

## 字數目標
- 總字數：1500-2000 字
- H2 數量：4-5 個
- 觀察平台：5-8 個
- 內鏈數量：3 篇
