# 📚 PILLAR 總結型模板
# 適用：總結權威文（6/29 詐騙完全指南）

## 結構規範

```
H1: 娛樂城[主題]完全指南：[副標]
  - 總結詞：完全指南、終極攻略、全面整理
  - 副標：「預防、識別、自救到風險管理」「從入門到精通」

引言（150-200字）：
  - 「這是本站關於...最完整的整理...」
  - 「無論你是新手還是有經驗的玩家...」
  - 「建議收藏，定期回來查看最新更新」
  - 標註「最後更新日期」

H2: 第一部分：預防
  - 事前如何判斷平台安全
  - 檢查清單（5-8 項）
  - 常見陷阱識別
  - 字數：500-700字
  - 內鏈：2-3 篇相關子文章

H2: 第二部分：識別
  - 常見徵兆 / 危險信號
  - 8-10 個 red flags
  - 每個徵兆配簡短說明
  - 字數：500-700字
  - 內鏈：2-3 篇相關子文章

H2: 第三部分：自救
  - 遇到問題怎麼辦
  - 處理流程（步驟 1/2/3）
  - 各管道聯繫方式
  - 字數：400-600字
  - 內鏈：2-3 篇相關子文章

H2: 第四部分：風險管理
  - 長期保護策略
  - 資金安全原則
  - 心態管理
  - 字數：300-500字

H2: 常見問題 FAQ（10 題速答）
  - 10 個最常見問題
  - 每題 2-3 句簡短回答
  - 字數：400-500字

H2: 延伸閱讀
  - 內鏈至所有相關子文章（10+ 篇）
  - 分類：品牌查證 / 教學 / 推薦
  - 字數：200-300字

結尾：免責聲明 + 「最後更新日期」標註 + 鼓勵收藏
```

## PILLAR 文特殊規範

1. **必為全站最長文章**：3000-4000 字
2. **必含 10+ 內鏈**：連結到所有相關子文章
3. **必含 FAQ Schema**：10 題 FAQ 結構化標記
4. **必標註更新日期**：每月更新一次
5. **必含目錄（Table of Contents）**：便於導航

## 內鏈策略（PILLAR 專用）

| 區塊 | 內鏈數 | 連結目標 |
|------|--------|---------|
| 預防 | 3 篇 | 安全檢查、註冊教學、如何判斷 |
| 識別 | 3 篇 | 詐騙手法、黑網解析、品牌查證×1 |
| 自救 | 3 篇 | 被騙怎麼辦、法律途徑、證據蒐集 |
| 風險管理 | 2 篇 | 資金管理、推薦文 |
| 延伸閱讀 | 10+ 篇 | 全站相關文章分類列表 |

## 字數目標
- 總字數：3000-4000 字（全站最長）
- H2 數量：6-7 個
- FAQ 數量：10 題
- 內鏈數量：15+ 篇

## Schema 標記

必含以下結構化資料：
- Article（文章）
- FAQPage（10 題 FAQ）
- BreadcrumbList（麵包屑）
- 最後更新日期（dateModified）

## 更新策略

每月 1 日檢查：
- [ ] 是否有新詐騙手法需加入
- [ ] 內鏈是否有效（無 404）
- [ ] 更新日期是否已更新
- [ ] 是否有新品牌查證文可加入延伸閱讀
