# 🏆 平台比較型模板
# 適用：推薦文（6/12）、排名文（6/19）

## 結構規範

```
H1: 2026年[月份][安全娛樂城推薦/排名]：[副標]
  - 正向詞：推薦、排名、比較、精選
  - 副標：「出金穩定與安全性比較」「合法平台篩選標準」

引言（100-150字）：
  - 「選擇娛樂城時，安全是最重要的考量...」
  - 「我們根據...標準整理以下...」
  - 「以下僅供參考，請自行評估」

H2: 我們的評比標準
  - 列出 4-5 個標準：
    - 合法性 / 牌照
    - 出金穩定性
    - 客服品質
    - 玩家評價
    - 遊戲種類
  - 每個標準 2-3 句說明
  - 字數：200-300字

H2: [月份]推薦平台詳細介紹
  - 每個平台：
    - H3: [平台名稱]
      - 特色介紹（2-3 句）
      - 優點：2-3 點
      - 缺點/注意事項：1-2 點（客觀）
      - 適合誰：新手/老手/重出金/重優惠
    - 3-5 個平台
  - 字數：800-1000字

H2: 不同需求玩家的選擇建議
  - 新手玩家 → 推薦 XX（原因）
  - 重出金速度 → 推薦 XX（原因）
  - 重優惠活動 → 推薦 XX（原因）
  - 重遊戲種類 → 推薦 XX（原因）
  - 字數：300-400字

H2: 選擇平台前的最後檢查清單
  - 5 項檢查要點（簡短列表）
  - 導流至安全教學文
  - 字數：150-200字

H2: 常見問題 FAQ
  - 3-5 個常見問題
  - Q: 這些平台安全嗎？
  - Q: 如何判斷平台是否合法？
  - Q: 出金速度怎麼樣？
  - 字數：200-300字

H2: 相關文章推薦（內鏈區）
  - 3 篇內鏈
  - 可含 1 篇風險觀察文（反面參考）

結尾：客觀聲明「評比僅供參考，請自行查證」
```

## 推薦 vs 排名差異化

| 維度 | 推薦文（6/12） | 排名文（6/19） |
|------|---------------|---------------|
| 結構 | 分類推薦（適合誰） | 排名順序（1st/2nd/3rd） |
| 平台數 | 3-4 個精選 | 5-8 個比較 |
| 表格 | 少，側重描述 | 多，側重數據比較 |
| H3 | 「適合新手的平台」 | 「第1名：XX娛樂城」 |
| CTA | 「立即試玩」 | 「查看完整比較」 |

## 用語檢查清單

- [ ] 有優點也有客觀缺點（不過度美化）
- [ ] 無「最安全」「絕對可靠」等絕對詞
- [ ] 每個推薦有「適合誰」說明
- [ ] 文末有「僅供參考」聲明

## 內鏈策略
- 1 篇相關推薦/排名（不同類型）
- 1 篇教學型（如何選擇）
- 1 篇品牌查證（反面參考）

## 字數目標
- 總字數：2000-2500 字
- H2 數量：5-6 個
- 推薦平台：3-5 個
- 內鏈數量：3 篇
