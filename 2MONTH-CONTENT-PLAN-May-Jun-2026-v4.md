### 驗證方式
所有日期與星期經 Python `datetime.weekday()` 計算確認：
```python
from datetime import date
for m in [5,6]:
    for d in range(1, 32 if m==5 else 31):
        dt = date(2026, m, d)
        if dt.weekday() in [0,2,4]:
            names = ['一','二','三','四','五','六','日']
            print(dt.strftime("%m/%d"), "週"+names[dt.weekday()])
```
