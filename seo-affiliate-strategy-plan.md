## 4. CTA 策略

### CTA 設計原則

| 原則 | 說明 |
|-----|------|
| 相關性 | CTA 與內容主題相關 |
| 價值感 | 強調優惠或好處 |
| 急迫感 | 時效性、限量感 |
| 信任感 | 附帶安全提示或評價 |
| 不強迫 | 不彈窗、不強制跳轉 |

### CTA 類型與位置

#### 主要 CTA（大按鈕）
```
位置：Money Pages 中段、評測頁中段
文字：
  ✅ 「立即註冊領取 $300 體驗金」
  ✅ 「免費試玩 + 首存 100% 加碼」
  ❌ 「點擊這裡」（無價值）
  ❌ 「立即賭博」（負面）

設計：
  - 顏色：對比色（橘色、綠色）
  - 大小：大於一般按鈕
  - 位置：視覺動線終點
```

#### 次要 CTA（連結型）
```
位置：文章內文、側邊欄
文字：
  - 「查看完整評測」→ 評測頁
  - 「比較更多平台」→ 比較頁
  - 「領取本月最佳優惠」→ 優惠頁
```

#### 軟性 CTA（內容型）
```
位置：文章結尾
文字：
  「想試試看？天下娛樂城目前提供【首存1000送1000】優惠，
   點擊這裡立即註冊，15分鐘內即可開始遊戲。」
```

### CTA 頻率控制

| 頁面類型 | CTA 數量 | 位置 |
|---------|---------|------|
| SEO 文章 | 1-2 個 | 文末或側邊 |
| 評測頁 | 2-3 個 | 中段、結尾、側邊 |
| Money Page | 3-5 個 | 各推薦項目下方 |
| 首頁 | 2-3 個 | Hero區、推薦區 |

## 6. SEO 與轉化率兼顧策略

### 核心原則：「先給後拿」

**Google 視角（SEO）：**
- 內容是否有價值？→ 是（詳細教學、評測）
- 是否過度廣告？→ 否（內容為主，廣告為輔）
- 用戶體驗如何？→ 好（資訊完整，無強迫推銷）

**用戶視角（轉化）：**
- 找到想要的資訊？→ 是（解決問題）
- 有推薦解決方案？→ 是（平台推薦）
- 推薦是否可信？→ 是（評測透明、多家比較）

### 平衡技巧

#### 1. 內容與 CTA 比例
```
✅ 好比例：
  文章內容：80%
  推薦 CTA：20%

❌ 壞比例：
  文章內容：20%
  廣告推銷：80%
```

#### 2. CTA 文案軟化
```
❌ 硬推銷：「立即賭博贏大錢！」
✅ 軟推薦：「想實際體驗？可以試玩看看，註冊送免費試玩金」

❌ 強迫式：「不點擊就錯過！」
✅ 建議式：「如果你對快速出金有需求，天下娛樂城是個不錯的選擇」
```

#### 3. 價值交換
```
給：詳細的百家樂看路技巧（價值）
拿：適時推薦實戰平台（轉化）

給：優惠比較表格（價值）
拿：點擊連結領取優惠（轉化）
```

#### 4. 頁面分流設計
```
【資訊型文章】（純 SEO）
  - 內容：90%
  - CTA：10%（文末軟性推薦）

【評測頁】（SEO + 轉化）
  - 內容：70%
  - CTA：30%（中段+結尾）

【Money Page】（純轉化）
  - 內容：50%（比較、評測摘要）
  - CTA：50%（推薦列表）
```
