## 色調建議

| 用途 | 色彩 | 色碼參考 |
|------|------|----------|
| **主色** | 深紅/暗紅 | #8B0000, #B22222 |
| **輔色** | 深灰黑 | #1A1A1A, #2D2D2D |
| **點綴色** | 警示金/橙 | #FF8C00, #DAA520 |
| **文字色** | 純白（主標）、亮黃（強調）| #FFFFFF, #FFD700 |
| **背景** | 深灰漸層到近黑 | #1A1A1A → #0D0D0D |

**配色邏輯**: 
- 深紅 + 黑色 = 危險、警示、嚴肅
- 金色點綴 = 誘餌、金錢誘惑（對應「分潤」主題）
- 避免全紅（刺眼），用深色底讓紅色更有重量感

### 構圖要點
1. **三分法則**: 標題位於左側 1/3 至中央，視覺元素位於右側 2/3
2. **視線引導**: 從左上標題 → 右下視覺元素，形成對角線動態
3. **留白**: 標題四周保留足夠呼吸空間（至少 60px 邊距）
4. **層次**: 背景（深）→ 視覺元素（中）→ 標題（前，高對比）

### 字體建議
| 層級 | 字體類型 | 大小（px）| 字重 |
|------|---------|----------|------|
| 主標 | 粗體黑體/無襯線 | 48-56 | Bold/Heavy |
| 副標 | 同主標 | 28-32 | Medium |
| 年份徽章 | 等寬或科技感 | 18-22 | Bold |

### 台灣繁體中文字體建議
- 首選：思源黑體 Heavy（Noto Sans TC Black）
- 替代：微軟正黑體粗體（若無授權問題）
- 英文/數字：Montserrat Bold / Roboto Bold

### 文字處理
- 主標題加「文字陰影」增加可讀性：`text-shadow: 0 2px 10px rgba(0,0,0,0.7)`
- 可加「細線框」強調：`-webkit-text-stroke: 1px rgba(255,255,255,0.3)`
- 「2026最新」使用對比色標籤（金色底 + 深色字）
