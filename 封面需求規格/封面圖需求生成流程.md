# fun1399 SEO 文章封面圖需求規格生成流程

## 流程概述

每次新文章完成後，自動執行以下步驟產出封面圖需求規格。

---

## 步驟 1：文章主題分析

### 自動提取資訊
- 文章標題
- 文章類型（判斷邏輯見下方）
- 核心關鍵字（取標題前 3-5 個名詞）
- 情緒調性（由類型推導）

### 文章類型判斷表

| 關鍵字特徵 | 類型 | 調性 | 主色傾向 |
|-----------|------|------|----------|
| 詐騙/陷阱/警示/風險/黑網 | 警示文 | 嚴肅、警示 | 深紅+黑 |
| 攻略/教學/指南/步驟/怎麼 | 教學文 | 實用、可信 | 藍+白 |
| 推薦/評價/比較/排行 | 評測文 | 客觀、比較 | 綠/金+白 |
| 優惠/體驗金/返水/首儲 | 優惠文 | 興奮、吸引 | 金+紅 |
| 2026/最新/趨勢/預測 | 時事文 | 新穎、前瞻 | 藍紫+白 |
| 新手/入門/基礎 | 入門文 | 親切、引導 | 綠+黃 |

### 分析輸出範例
```
文章: 娛樂城代理詐騙：拉人頭分潤的傳銷陷阱解析（2026最新）
類型: 警示文（關鍵字：詐騙、陷阱、風險）
調性: 嚴肅、警示、專業
主色: 深紅+黑+金
```

---

## 步驟 2：視覺元素建議（依類型自動匹配）

### 警示文（詐騙/陷阱/黑網）
**意象池**: 鎖鏈、蜘蛛網、陷阱、警告標誌、面具、金字塔、斷裂的橋、深淵
**輔助元素**: ⚠️ 三角形、紅色印章、警戒線、裂縫
**參考風格**: 新聞專題封面、調查報導

### 教學文（攻略/指南/步驟）
**意象池**: 指南針、地圖、路標、燈塔、書本、清單、流程圖
**輔助元素**: 數字標籤（Step 1/2/3）、打勾符號、書籤
**參考風格**: 教育平台、知識付費課程封面

### 評測文（推薦/比較/排行）
**意象池**: 天平、獎盃、冠軍台、星等、盾牌、認證徽章
**輔助元素**: TOP 1/2/3 標籤、對勾、金邊框
**參考風格**: 產品評測雜誌、消費者報告

### 優惠文（體驗金/返水/首儲）
**意象池**: 金幣雨、禮物盒、紅包、煙火、慶祝彩帶、閃光
**輔助元素**: 「限量」「限時」「獨家」標籤、倒數計時
**參考風格**: 電商促銷 Banner、金融機構活動頁

### 時事文（2026/最新/趨勢）
**意象池**: 日曆翻頁、時鐘、趨勢箭頭、數據圖表、未來城市
**輔助元素**: 「2026」「NEW」「更新」徽章
**參考風格**: 財經媒體、科技新聞

### 入門文（新手/基礎）
**意象池**: 門/入口、階梯、握手、引導線、溫暖燈光
**輔助元素**: 「新手必讀」「零基礎」標籤、柔和圓角
**參考風格**: 線上學習平台、入門書籍封面

---

## 步驟 3：色調自動匹配

### 配色邏輯矩陣

| 類型 | 主色 | 輔色 | 點綴色 | 背景 |
|------|------|------|--------|------|
| 警示文 | 深紅 #8B0000 | 黑 #1A1A1A | 金 #DAA520 | 深灰漸層 |
| 教學文 | 深藍 #1E3A5F | 白 #FFFFFF | 亮藍 #4A90E2 | 淺灰藍 |
| 評測文 | 墨綠 #2D5016 | 米白 #F5F5F0 | 金 #DAA520 | 淺灰 |
| 優惠文 | 亮紅 #FF6B35 | 金 #FFD700 | 白 #FFFFFF | 深紅/金漸層 |
| 時事文 | 紫藍 #4A369D | 白 #FFFFFF | 青 #00D4AA | 深藍灰 |
| 入門文 | 青綠 #00A86B | 白 #FFFFFF | 黃 #FFD93D | 淺綠灰 |

### 配色邏輯說明
- **主色** = 文章情緒的代表色（紅=危險/興奮，藍=可信，綠=安全/推薦）
- **輔色** = 主色的對比或中性色，確保文字可讀
- **點綴色** = 高亮重點（標籤、徽章、CTA）
- **背景** = 承托主色，深色底適合警示/優惠，淺色底適合教學/評測

## 步驟 4：構圖自動配置

### 通用構圖規則（所有類型適用）

```
[1200 x 630 畫面分區]

┌─────────────────────────────────────────┐
│ [年份/類型標籤]              [品牌水印]  │  ← 10% 頂部邊距
│                                         │
│  ┌─────────────────┐  ┌──────────────┐ │
│  │                 │  │              │ │
│  │   標題文字區    │  │  視覺元素區  │ │  ← 60% 核心區
│  │   (左側 55%)    │  │  (右側 45%)  │ │
│  │                 │  │              │ │
│  └─────────────────┘  └──────────────┘ │
│                                         │
│         [漸層遮罩/底部裝飾]              │  ← 20% 底部
└─────────────────────────────────────────┘
```

### 文字安全區
- 社交媒體（Facebook/Line）裁切規則：中間 800x400 為安全區
- 標題文字必須完全落在安全區內
- 視覺元素可部分超出（被裁切不影響理解）

---

## 步驟 5：標題文字自動處理

### 字體規範（固定）
| 層級 | 字體 | 大小 | 字重 | 效果 |
|------|------|------|------|------|
| 主標 | Noto Sans TC Black | 48-56px | Heavy | 陰影+描邊 |
| 副標 | Noto Sans TC Bold | 28-32px | Bold | 陰影 |
| 徽章 | Montserrat Bold | 18-22px | Bold | 底色塊 |

### 文字處理規範（固定）
- 主標題加陰影：`0 2px 10px rgba(0,0,0,0.7)`
- 可加細線框：`-webkit-text-stroke: 1px rgba(255,255,255,0.3)`
- 限制字數：主標 < 12 字，副標 < 20 字（超出則縮小字體或分行）

---

## 步驟 6：輸出交付

### 自動產出兩份文件

**A. 完整規格文件**（給設計師/AI 詳細使用）
```
封面需求規格/{slug}-cover-spec.md
```

**B. 簡易摘要**（快速參考）
```
封面需求規格/{slug}-cover-brief.md
```

### 檔案命名規則
```
{article-slug}-cover.jpg
```
範例：`casino-agent-scam-2026-cover.jpg`

## 完整執行範例

### 輸入
```
文章標題：娛樂城代理詐騙：拉人頭分潤的傳銷陷阱解析（2026最新）
```

### 自動分析
1. **類型判斷**: 標題含「詐騙」「陷阱」→ 警示文
2. **調性**: 嚴肅、警示、專業
3. **主色**: 深紅+黑+金（警示文配色）
4. **視覺元素**: 鎖鏈/陷阱/面具（警示文意象池）
5. **構圖**: 左文右圖，標題置中偏上

### 自動輸出
- `封面需求規格/casino-agent-scam-2026-cover-spec.md`（完整版）
- `封面需求規格/casino-agent-scam-2026-cover-brief.md`（摘要版）

## 與發文流程整合

### 新文章完整流程

```
1. 產出文章 HTML
2. 更新首頁/分類頁/sitemap
3. 【新增】產出封面圖需求規格（本流程）
4. 建立 Netlify Preview Deploy
5. 回報 Preview URL + 封面規格檔案路徑
6. 【等待 Hans 確認】
7. 收到封面圖後 → 壓縮/轉 WebP → 上傳 static/images/
8. 更新文章 HTML 的 og:image / twitter:image / hero image
9. 重新 Preview Deploy
10. 【等待 Hans 確認「正式部署」】
11. Production Deploy
```
