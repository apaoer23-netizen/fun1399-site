### 驗證方式
所有日期與星期經 Python `datetime.weekday()` 計算確認：
```python
from datetime import date
for m in [5,6]:
    for d in range(1, 32 if m==5 else 31):
        dt = date(2026, m, d)
        if dt.weekday() in [0,2,4]:  # 0=Mon, 2=Wed, 4=Fri
            print(dt, "週一三五"[dt.weekday()//2])
```
