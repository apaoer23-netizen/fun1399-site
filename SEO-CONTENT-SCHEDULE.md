## 🔄 兩階段部署流程

### 第一階段：Preview（自動執行）

```
1. 分析現有文章
2. 選擇新主題
3. 生成文章（含完整 author-info）
4. 添加內部連結
5. 添加延伸閱讀
6. 更新 sitemap
7. Build 網站
8. ⭐ 等待用戶確認
```

**回報內容：**
- 文章標題
- 目標關鍵字
- 文章分類
- 內鏈數量
- Build 狀態
- 檔案路徑

### 第二階段：正式部署（等待確認）

**觸發條件：** 用戶回覆「可以上線」

```
9. 部署到 Netlify（正式環境）
10. 回報部署完成
```

---

## 🎯 手動執行預覽部署

如需手動部署預覽版本：

```bash
# 進入網站目錄
cd /root/.openclaw/workspace/fun1399-site

# Build（如需要）
# npm run build

# 部署 Preview 版本
netlify deploy --dir=build --draft

# 輸出 Preview URL
```

---

## 🎯 手動執行正式部署

**僅在用戶確認「可以上線」後執行：**

```bash
# 進入網站目錄
cd /root/.openclaw/workspace/fun1399-site

# 部署到正式網站
netlify deploy --dir=build --prod
```
