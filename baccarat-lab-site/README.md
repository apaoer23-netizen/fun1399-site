## 🖥️ 預覽方式

### 方法一：直接開啟（最簡單）

1. 解壓縮 ZIP 檔案
2. 雙擊開啟 `index.html`
3. 使用瀏覽器預覽（Chrome/Safari/Edge）

### 方法二：本地伺服器（推薦）

```bash
# 進入網站目錄
cd baccarat-lab-site

# Python 簡易伺服器
python3 -m http.server 8080

# 瀏覽器開啟
http://localhost:8080
```

## 🎯 設計重點

### 高轉換設計特色

1. **Sticky CTA** - 頁面滾動時持續顯示的行動按鈕
2. **多點導流** - 每篇文章至少3個CTA（文中/側邊/結尾）
3. **緊迫感元素** - 限時優惠、剩餘名額、倒數計時
4. **信任背書** - 實戰數據、出金證明、玩家評價
5. **LINE私域** - 多處引導加LINE，建立私域流量

### RWD響應式斷點

- **桌機**：1200px+ （完整版面）
- **平板**：768px - 1024px （調整為2欄）
- **手機**：768px以下 （單欄，Sticky CTA優化）
