## 技術筆記

### ImageMagick 封面圖生成指令範例
```bash
convert -size 1280x800 xc:none \
  -fill "#1a1a2e" -draw "roundrectangle 0,0 1280,800 20,20" \
  \( -size 1280x800 gradient:#2d2d5a-#1a1a2e \) -compose DstIn -composite \
  -pointsize 60 -fill "#ff6b35" -gravity center -annotate +0,-100 "標題" \
  -pointsize 40 -fill "#ffffff" -gravity center -annotate +0,-20 "副標題" \
  output.png
```

### 對比度修復 CSS 模式
```css
background: linear-gradient(135deg, #2a2a4e 0%, #1e3a5f 100%);
color: #ffffff; /* 或 #cccccc 用於次要文字 */
```
