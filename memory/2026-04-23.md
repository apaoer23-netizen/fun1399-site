## 2026-04-23 22:25 更新：.html → 無副檔名 URL 301 Redirect 修正

### 問題
- /articles/hg-casino-review.html 與 /articles/hg-casino-review 同時存在
- Google 索引失敗（重複內容）

### 修正內容

**1. _redirects 文件**
- 原因：wildcard `/articles/*.html → /articles/:splat` 的 `:splat` 會帶入 `.html`，導致重定向到同一 URL
- 解決：改為逐一列出所有文章 redirect（共 67+ 條）
- 語法：`/articles/xxx.html /articles/xxx 301!`
- `!` = force，即使 .html 檔案存在也強制重定向

**2. sitemap.xml**
- 移除所有 `.html` 結尾
- 從 68 個帶 .html 的 URL → 0 個

**3. 全站內鏈**
- 修正所有 `href=".../xxx.html"` → `href=".../xxx"`
- 包含 articles/ 內所有文章、index.html

**4. canonical URL**
- 修正 3 篇新文章的 canonical（移除 .html）
