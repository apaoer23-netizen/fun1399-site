## 🔧 SEO 修復：深色背景文字對比（2026-05-20 13:22-14:15）

### 問題
- CTA 區塊與 4 張優惠卡片內文仍為黑色（#111），在深色背景上幾乎不可見
- 根子因：`.article-content p { color: #111 !important; }` 的全局 CSS 覆蓋了所有 inline color

### 修正內容
1. **移除全局 CSS 的 `!important`**：`.article-content p` 不再強制 #111
2. **深色卡片專用 CSS**：新增 `.cta-box p, div[style*="#1a1a2e"] p` 等選擇器，強制 `color: #e5e7eb !important`
3. **補充 `!important` 到 inline styles**：所有 dark card 的 p、h3、link 都加上 `!important`
